Sistema de estudos focado em organização, planejamento e evolução do aprendizado.

**Stack:** Python, FastAPI, PostgreSQL, Next.js, Docker e Alembic

**Conceitos:**
- Arquitetura fullstack
- Autenticação
- CRUD
- Dashboard
- Planner de estudos
- Organização modular
- Integração frontend + backend

**Objetivo:** construir uma plataforma de estudos útil, organizada e escalável, usando o projeto como laboratório real de desenvolvimento.

- **Deploy:** [StudyOS](https://study-os-20-oldkyzi6h-erik02ts-projects.vercel.app)

### Anime Manager
Aplicação para gerenciamento de animes com foco em organização, progresso e experiência do usuário.

**Stack:** React, TypeScript, Tailwind, FastAPI e PostgreSQL

**Conceitos:**
- Cadastro e gerenciamento de itens
- Filtros e organização
- Estado da aplicação
- Comunicação com API
- Estrutura frontend/backend
- Produto visual e funcional

**Objetivo:** criar um sistema atrativo e útil enquanto aprofundo conhecimentos em fullstack e arquitetura de produto.

- **Deploy:** [Anime Manager](https://anime-manager-web.vercel.app/login)
- **Frontend:** [anime-manager-web](https://github.com/Erik02T/anime-manager-web)
- **Backend:** [anime-manager-api](https://github.com/Erik02T/anime-manager-api)
